## 项目介绍
基于JSP+Servlet实现的高校社团管理系统系统，主要功能如下

【前台功能】
首页、社团列表、社团风彩、社团活动、

【后台功能】

个人中心：密码修改
系统管理：用户管理、新闻管理、留言管理
社团团里：社团列表管理、社团风彩管理、社团活动管理

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、JavaScript、bootstrap、JQuery
- 后端技术：Servlet、JDBC

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL5.7及以上
